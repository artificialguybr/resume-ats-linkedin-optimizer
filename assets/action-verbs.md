# Action Verb Bank

Start every bullet with a strong, **plain** verb. Prefer concrete verbs that
describe what actually happened over buzzword-inflated ones (see
`ai-slop-blacklist.md` — words like *spearheaded, orchestrated, leveraged* now
read as AI slop; use the plain equivalent).

## By category (Harvard / MIT career services)

**Leadership / management:** Led, Directed, Managed, Coordinated, Oversaw,
Organized, Supervised, Administered, Planned, Delegated, Mobilized, Drove.

**Communication:** Presented, Wrote, Authored, Delivered, Negotiated, Promoted,
Collaborated, Addressed, Persuaded, Briefed, Documented.

**Creation / building:** Built, Designed, Created, Developed, Launched, Founded,
Established, Introduced, Produced, Shipped, Prototyped, Implemented.

**Improvement / impact:** Increased, Grew, Improved, Boosted, Accelerated,
Reduced, Cut, Slashed, Eliminated, Saved, Streamlined, Optimized, Automated,
Doubled, Tripled, Exceeded, Surpassed, Delivered, Generated.

**Analysis / research:** Analyzed, Evaluated, Investigated, Identified, Assessed,
Measured, Forecasted, Modeled, Researched, Diagnosed, Audited.

**Technical:** Engineered, Programmed, Coded, Deployed, Integrated, Configured,
Migrated, Debugged, Scaled, Refactored, Instrumented, Tested.

**Finance / quantitative:** Budgeted, Allocated, Forecasted, Projected, Reduced
(costs), Reconciled, Modeled, Managed (P&L).

**Teaching / support:** Trained, Coached, Mentored, Advised, Facilitated,
Instructed, Onboarded, Guided, Supported.

**Organization / execution:** Implemented, Executed, Processed, Arranged,
Completed, Standardized, Consolidated, Verified.

## Verbs to avoid or downgrade

- **Filler / weak:** Responsible for, Duties included, Helped, Assisted with,
  Worked on, Handled, Tasked with, In charge of, Participated in, Familiar with.
- **Inflated / AI-tell (prefer the plain form):** Spearheaded → led ·
  Orchestrated → coordinated · Architected → designed · Leveraged → used ·
  Utilized → used · Championed → advocated for · Pioneered → introduced ·
  Facilitated → ran/helped · Harnessed → used · Drove (metaphorical overuse).

## Rules

- One verb per bullet, past tense (present for the current role).
- Never start a bullet with a noun, a date, or a pronoun.
- Match the verb to a real, quantified outcome — a strong verb on an empty claim
  fools no one.
